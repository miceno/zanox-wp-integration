Allows the load Zanox HTML affiliate code.

== Description ==

This plugin adds a new widget to the sidebar which can show the different available Zanox widgets showing advertisers programmes.

== Installation ==

1. Upload `zanox-widget.php` to the `/wp-content/plugins/` directory
1. Place the widget into your sidebar using the 'Presentation' menu in WordPress
1. Adjust the widget's options so that it can show some useful content
1. Save changes. ;)
